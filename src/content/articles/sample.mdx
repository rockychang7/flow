# 1. width使用方式

- Measurements：width支持 100%，200px,5rem等固定大小的方式定义width
- Keywords: 可以通过关键字定义width，不同关键字自适应的width策略各不相同
```css
width:auto; //默认值，auto会尽量达到可获得空间的最大width(受margin,padding等影响)

width: min-content; //min-content会将width设置为能够容纳最长的那个children content的宽度，当他遇到空白符或者连接符都会换新行

width: max-content;  //max-content会将width设置容纳整个content的宽度，并且超出parent container也不会换行

//1. 如果content的内容没有超出parent的宽度，那么就表现为max-content的行为
//2. 如果content的内容超出了parent的宽度，那么就表现为auto的行为
width: fit-content;
```

## max-width和min-width
通常用来定义元素的最大和最小的宽度

```css
 .box {
    width: 50%;
    min-width: 170px;
    max-width: 300px;
    border: solid hotpink;
  }
```
